# @advjs/gui

GUI for ADV.JS Engine.

## Usage

```ts
// icons css vars
import '@advjs/blender-ui/dist/icons.css'
// common css
import '@advjs/blender-ui/client/styles/index.scss'
```

### Nuxt

```ts
// nuxt.config.ts
export default defineNuxtConfig({
  modules: [
    // ...
    '@advjs/gui/nuxt',
  ],
})
```

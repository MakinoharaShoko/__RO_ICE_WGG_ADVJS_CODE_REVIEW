# @advjs/gui

GUI for ADV.JS Engine.

## Usage

```ts
// icons css vars
import '@advjs/gui/dist/icons.css'
// common css
import '@advjs/gui/client/styles/index.scss'
```

### Nuxt

```ts
// nuxt.config.ts
export default defineNuxtConfig({
  modules: [
    // ...
    '@advjs/gui/nuxt',
  ],
})
```

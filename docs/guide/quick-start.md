## Local

执行以下命令，以在本地创建一个 advjs 项目模版。

```bash
pnpm create adv@latest your-adv
```

With NPM:

```bash
npm init adv
```

With Yarn:

```bash
yarn create adv
```

进入目录并启动：

```bash
cd your-adv
pnpm i
pnpm run dev
```
